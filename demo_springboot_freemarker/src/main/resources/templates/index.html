  • ${(item.userName)!?html},的生日是${item.birthDay!?string('YYYY-MM-DD HH:mm:ss')}
  • 整数：${intVal!}
    长整数：${longVal!}
    字符串：${stringVal!}
    双精度：${doubleVal!}
    布尔值：${booleanVal!?string('yes','no')}
    日期：${dateVal!?string('YYYY-MM-DD HH:mm:ss')}
    null值：${nullVal!}
    null值：${nullVal!'我是默认值'}
    miss值：${missVal!'我是丢失默认值'}
  • 赋值&&运算
    • <#assign a = 100 /> a = ${a}
    a + 100 = ${a + 100}
  • 循环遍历map数据
    • <#list map?keys as key> ${key}:${map[key]}

    • <#assign var = 99 /> <#if var == 99> var = ${var}
    <#if var == 100> var = ${var}
    <#else> var != 100
    <#if var > 100> var 大于 ${var}
    <#elseif var < 100> var 小于 100
    <#else> var = 100

    • <#assign vars = 'java' /> <#if vars == 'python' || vars == 'java'> vars = ${vars}
    <#if vars == 'java' && vars?length == 4> vars = ${vars}
